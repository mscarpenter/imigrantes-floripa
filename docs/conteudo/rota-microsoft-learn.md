## 1. Por que Microsoft Learn neste portal

| Critério | Escola Virtual Bradesco (EV) | Microsoft Learn |
|----------|------------------------------|-----------------|
| Custo | Gratuito | Gratuito (módulos e roteiros) |
| Cadastro | Conta EV (**exige CPF**) | Conta Microsoft (e-mail), **sem CPF** |
| Idioma | Só português (Brasil) | **65+ locales** (pt-br, es-es, en-us, fr-fr e mais) |
| Certificado | Gratuito na EV (≥ 70%) | Troféus/conquistas no perfil Learn; certificações Microsoft são pagas |
| Relevância | Forte no mercado brasileiro | Global; conteúdo oficial Microsoft/GitHub |

**Posicionamento editorial:** alternativa imediata para quem ainda não tem CPF. Complementa (não substitui) a rota EV Bradesco quando o documento estiver regularizado.

## 5. Como se inscrever

1. Acesse [learn.microsoft.com](https://learn.microsoft.com/pt-br/) no idioma desejado (rodapé).
2. Clique em **Entrar** (canto superior direito).
3. Use uma **conta Microsoft pessoal** (Outlook, Hotmail ou crie uma com qualquer e-mail).
4. Na primeira vez, configure o **perfil Learn** (nome de exibição; campos fiscais são opcionais).
5. Abra um roteiro da lista acima e clique em **Iniciar** em cada módulo.
6. O progresso fica salvo no perfil; troféus aparecem ao concluir módulos.

**Sem CPF:** todo o fluxo acima funciona sem documento brasileiro.

**Com CPF depois:** considere também a [Escola Virtual Bradesco](/pt/blog/cursos-gratuitos-tecnologia-escola-virtual-bradesco), com certificados gratuitos reconhecidos localmente.

## 6. Limitações honestas

- Certificações Microsoft (ex.: AZ-900, PL-300) exigem **exame pago**; os roteiros Learn são preparatórios gratuitos.
- Nem todo roteiro está 100% traduzido; alguns módulos podem abrir em inglês.
- GitHub Copilot avançado pode exigir assinatura (há plano gratuito para perfis elegíveis).
- Laboratórios Azure gratuitos têm limites de tempo/uso.
